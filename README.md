# Hadoop-searchEng

## Pre-requisites
```bash
pip install -r requirements.txt
```

## Setting Dockers
```
# make
```
## Delete dockers
```
# make clean
```
## How to Search

Inside terminal:

```
# python3 search/search.py
```

```
----- Escribir .exit para salir -----

```
